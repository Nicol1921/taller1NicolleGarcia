UN POCO DE JAVASCRIPT

JavaScript es un lenguaje de programación popular que se utiliza principalmente para agregar interactividad a los sitios web. Con JavaScript, puedes crear efectos dinámicos, manipular el contenido de la página, interactuar con el usuario y mucho más. Es un lenguaje versátil y potente que se ejecuta en el navegador del usuario.

SU HISTORIA

JavaScript es un lenguaje de programación de alto nivel que se utiliza principalmente en el desarrollo web. Fue creado por Brendan Eich en 1995 mientras trabajaba en Netscape Communications Corporation (ahora conocida como Mozilla Corporation). Nacimiento de JavaScript: En 1995, Netscape estaba desarrollando el navegador Netscape Navigator y quería agregar capacidades interactivas a las páginas web.
