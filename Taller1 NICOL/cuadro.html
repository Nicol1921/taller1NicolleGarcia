¡10 EVENTOS DE JAVASCRIPT MÁS COMUNES!


EVENTO
 DESCRIPCIÓN DE APLICACIÓN
onclick Se activa cuando se hace clic en un elemento HTML.
onload Se dispara cuando se completa la carga de un elemento, como una página web o una imagen.
onkeydown Ocurre cuando se presiona una tecla.
onkeyup Se activa cuando se suelta una tecla.
onmouseover Se desencadena cuando el cursor del mouse se mueve sobre un elemento.
onmouseout Ocurre cuando el cursor del mouse se mueve fuera de un elemento.
onchange Se dispara cuando el valor de un elemento cambia, como un campo de entrada.
onsubmit Se activa cuando se envía un formulario.
onscroll Ocurre cuando el contenido de un elemento desplazable se desplaza.
onfocus Se dispara cuando un elemento obtiene el foco.
